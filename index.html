Podstawowe polecenia GITa

  • $git init
  • $git status
  • $git add plik
  • $git log --oneline
  • $git gitk
  • $git branch newBranch
  • $git checkout newBranch
  • $git branch -d newBranch
  • $git merge newBranch - Jak jesteśmy na gałęzi master(głównej) scali nam ją z gałęzią newBranch
  • $git remote add origin newBranch git@github.com:Nazwa/nazwa.git
  • $git push -u origin
  • $git push -u origin master newBranch [tu bierzemy dwie gałęzie master i nowa]
  • $git clone git@github.com:Nazwa/nazwa.git
  • $git clone -b newBranch git@github.com:Nazwa/nazwa.git
